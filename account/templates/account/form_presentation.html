A user mention

If you want to mention a user, use @<username> in the href attribute which is inside the a tag. You can also use Bootstrap class. Example:

<a href=@user ></a>

If you want to use a different style attribute. You have to place it after href attribute. You can also use Bootstrap class. Example:

<a href=@user class="text-uppercase" ></a>

A user profile photo mentions

If you want to mention a user profile photo, use @prof-<username > in the src attribute which is inside the img tag.

<img href=@user>

If you want to use a different style attribute. You have to place it after src attribute. You can also use Bootstrap class. Example:

<img href=@user width="500" height="600">
